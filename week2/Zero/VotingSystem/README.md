基于 ERC20 代币持有权的去中心化投票系统，核心逻辑是「只有持有指定 ERC20 代币的地址能投票，且每个地址仅能投 1 次」，同时加入了时间管控、管理员权限等实用功能

权限：基于Ownable实现管理员（合约部署者）专属操作（如添加候选人、修改投票时间）；
校验：多重安全校验（防止重复投票、无效候选人投票、非投票时段投票、无代币投票）；
可追溯：通过事件记录所有关键操作（添加候选人、投票、时间修改）；
灵活性：支持设置投票开始 / 结束时间，甚至紧急结束投票。

updateVotingTime(uint256 _newStartTime, uint256 _newEndTime)    修改投票开始 / 结束时间（比如测试时想延长投票时间） 仅管理员
emergencyEndVoting()    紧急结束投票（比如测试完成后手动终止）  仅管理员
isVotingActive()    查询当前投票是否处于激活状态（返回 true/false） 任何人
